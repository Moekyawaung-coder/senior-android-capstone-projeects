# Compose Design System — Senior Level Project

This is the first capstone project in my Senior Android Roadmap.

**Features Implemented:**
- 100% Jetpack Compose + Material 3
- Custom color tokens, typography scale, spacing system
- Dark mode with dynamic colors
- Reusable Button, Card, TextField, Loading indicators
- Full theming engine that can be published as library
- Baseline Profiles for instant startup

**Architecture:** Multi-module with :core:ui, :core:designsystem, :feature:demo
